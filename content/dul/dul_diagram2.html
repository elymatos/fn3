DOLCE Ultra Lite (DUL) Ontology

Main Classes and Relations - Foundational Framework for Reality Modeling

🌟 ENTITY
Entity
Root class: anything that can be talked about
🔵 OBJECTS
Object
Physical, social, or mental objects
PhysicalObject
Objects with spatial location
Agent
Objects that can act intentionally
PhysicalAgent
Physical objects capable of planning
Person
Human individuals
Organism
Biological objects
PhysicalArtifact
Designed physical objects
🟡 EVENTS
Event
Physical, social, or mental processes
Action
Events with intentional agents
Process
Events in their evolution
Situation
Contexts/views on sets of entities
PlanExecution
Situations satisfying plans
Transition
State change situations
🟣 SOCIAL
SocialObject
Objects existing through communication
Description
Conceptual frameworks
Plan
Descriptions with explicit goals
Concept
Ideas defined in descriptions
Role
Concepts that classify objects
Task
Event types for actions
InformationObject
Pieces of information content
Organization
Structured social agents
🟢 ABSTRACT
Abstract
Entities outside space-time
FormalEntity
Mathematical/logical entities
Collection
Containers for entities
🟠 QUALITIES
Quality
Aspects of entities
Region
Dimensional spaces/values
TimeInterval
Temporal regions
SpaceRegion
Spatial regions
Parameter
Constraints on regions
