# Ontologia das Relações na FN5

Este documento descreve formalmente os tipos de relações entre frames na FN5, incluindo suas categorias e definições.

| Relação                         | Categoria                                    | Definição |
|----------------------------------|-----------------------------------------------|-----------|
| **rel_inheritance** | Ontológica/Hierárquica | O frame A é um subtipo mais específico do frame B, herdando seus elementos de frame (EFs) e estrutura. |
| **rel_subframe** | Composicional/Eventiva | O frame A é um subevento ou subprocesso do frame B. Instanciar B requer a ocorrência de A. |
| **rel_perspective_on** | Focalização/Perspectiva | O frame A representa uma perspectiva ou foco específico sobre o frame B. |
| **rel_using** | Dependência Funcional | O frame A faz uso da estrutura conceitual ou dos EFs do frame B para ser interpretado. |
| **rel_precedes** | Sequencial/Temporal | O frame A ocorre sequencialmente antes de B em uma ordem temporal ou processual. |
| **rel_structural_causation** | Causal/Estrutural | O frame A representa a causa, fundamento ou condição estrutural necessária para a ocorrência do frame B. |
| **rel_structural_path** | Trajetória/Espaço/Processo | O frame A representa um ponto na trajetória (origem, meio ou destino) dentro da estrutura do frame B. |
| **rel_structural_containment** | Contenção/Espaço/Hierarquia | O frame A está contido estrutural ou conceitualmente dentro do frame B (parte de, subconjunto, etc.). |
| **rel_structural_link** | Conexão/Estrutural | O frame A está ligado funcional ou semanticamente ao frame B, representando conexão ou dependência. |
| **rel_structural_sequence** | Sequência/Estrutura/Processo | O frame A faz parte de uma sequência estrutural ordenada, na qual B é o próximo item. |
| **rel_structural_dependency** | Dependência Funcional ou Composicional | A existência ou validade do frame A depende funcionalmente da existência ou configuração do frame B. |
| **rel_interpretative_cooccurrence** | Coocorrência Semântica/Pragmática | A interpretação de A é fortalecida, dependente ou normalmente acompanhada pela ativação de B. |
| **rel_mutual_exclusion** | Lógica/Oposicional | A ocorrência do frame A exclui logicamente a de B. |
| **rel_alternative** | Alternativa/Escolha | A e B são alternativas possíveis dentro de uma mesma situação ou escolha. |
| **rel_participates_in** | Participação/Estrutural | O frame A representa uma entidade, qualidade ou estado que participa como elemento estrutural no frame B. |
| **rel_requires_precondition** | Condicional/Estrutural | A ocorrência de A exige que B esteja previamente satisfeito como pré-condição. |
